# 📘 ملاحظات المشروع

توثيق سريع للقرارات التقنية. كل سطر يجاوب على "ليش".

---

## 📦 package.json

### Scripts
- `npm start` — يشغّل السيرفر في الإنتاج (بدون مراقبة)
- `npm run dev` — يشغّل السيرفر بـ nodemon (يعيد التشغيل عند أي تعديل)
- `npm test` — placeholder للاختبارات لاحقاً

### Dependencies (المكتبات)

| المكتبة | الفائدة |
|---------|---------|
| **express** | يستقبل الريكوست وبرد عليها بـ ردود من فريم وورك اكسبرس |

| **prisma** + **@prisma/client** | ORM يمنع SQL Injection (متطلب صريح) |

| **jsonwebtoken** |(ختم) إنشاء/التحقق من JWT tokens |

| **bcrypt**  |لتشفير كلمات السر| 

| **cors** |يسمح للفرونت اند يتكلم مع الباك اند|

| **dotenv** | .env يقرأ المتغيرات الحساسة من ملف  |

| **zod** | يتحقق من بيانات المستخدم قبل ما نستخدمها |

### DevDependencies
- **nodemon** — يعيد تشغيل السيرفر تلقائيا وقت التطوير

##  ملاحظة أمنية

فيه 3 ثغرات moderate في `@hono/node-server` (مكتبة فرعية لـ Prisma Studio فقط).
**ما صلّحناها** لأن:
1. `audit fix` حذف Prisma كامل (180 package!)
2. `audit fix --force` يكسر المشروع
3. الثغرة ما تأثر على كودنا (نستخدم Express مو Hono)
